# Wordly Dictionary SPA

## Description

A Single Page Application that allows users to search for English words using the Free Dictionary API.

## Features

- Search for words
- View definitions
- View pronunciation
- Listen to pronunciation audio
- View synonyms
- Error handling
- Responsive interface

## Technologies

- HTML
- CSS
- JavaScript
- Fetch API
- Dictionary API

## API

https://dictionaryapi.dev/

## Installation

Clone the repository.

Open index.html in your browser.
